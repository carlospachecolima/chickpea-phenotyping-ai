AI-Assisted Chickpea Phenotyping

A prompt-engineering-based system for AI-assisted visual evaluation of physiological disorders in chickpea (Cicer arietinum), incorporating phenological context (Days After Planting - DAP).

Objective

To develop a digital phenotyping pipeline that integrates:

- Phenological context (DAP)
- AI-assisted visual analysis
- Automated diagnosis of plant stress and development

Methodological Approach

The system is based on structured prompt chaining:

1. Agronomic expert persona definition  
2. Phenological context injection (DAP-based interpretation rules)  
3. Guided visual observation  
4. Dynamic performance ranking (age-aware evaluation)

Key Innovation

This system explicitly incorporates **phenological stage (DAP)** into AI reasoning, allowing:

- Differentiation between natural senescence and pathological stress  
- Reduction of false positives in plant health diagnosis  
- Context-aware evaluation of plant development
